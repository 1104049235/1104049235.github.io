html 基础

基础框架

<!DOCTYPE html>   <!--标识文档类型 -->
<html lang="en">
<head>
	<meta charset="UTF-8" > <!--设置字符集 -->
	<meta name="viewport" content="width=device-width,initial-scale=1.0" >
	<meta http-equiv="X-UA-Compatible" content="ie=edge">
	<title>页面标题</title>
</head>
<body>
</body>
</html>

标签

<h1> <!--标题标签 -->
<section> <!--章节标签 -->
<a> <!-- 超链接 -->
<p> <!-- 段落 -->
<ol><li> <!--order list , list item -->

<dl><dt><dd> <!--description list,description title ,description data  -->

<br> <!-- 换行 -->

<strong> <!-- 强调 -->
<quote> <!-- 引用 --> 
<blockquote> <!-- 块引用 -->

全局属性

class

  • 标识类型名字
  • 这只css 属性

contenteditable

  • contenteditable=“true”
  • 可被修改 （加入该属性后html内容可被修改）

hidden

  • hidden
  • 标识该标签是否显示

id

  • 全局唯一标识 用来找到标签
  • 该标识如果重复html 不会报错提示

style

  • 设置该标签的样式内容

tabindex

  • 默认值为0
  • 大于0 标识可被聚焦 可以通过键盘导航找到该元素
  • 小于0 表示可以被聚焦 但是无法通过键盘访问该元素

title

  • 设置标签标题
  • 当该属性有值时鼠标放入会浮动显示
